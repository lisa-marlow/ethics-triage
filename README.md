# Ethics Triage Tool (Decision Support)

This repository contains a **static, browser-based ethics triage tool** developed at **Southern Cross University** to assist researchers and administrators in identifying the *most likely* review pathway for a proposed activity.

The tool provides **decision support only**. Final determinations rest with the relevant ethics committee or institutional authority.

---

## What this tool does

- Guides users through a structured series of questions
- Provides indicative outcomes based on responses
- References relevant sections of the *National Statement on Ethical Conduct in Human Research (2025)* and *Ethical Considerations in Quality Assurance and Evaluation Activities (2014)*
- Helps identify when ethics review, exemption assessment, or institutional oversight may be required

---

## What this tool does *not* do

- It does **not** provide legal advice
- It does **not** replace formal ethics review or institutional decision-making
- It does **not** store, transmit, or collect any user information
- It does **not** record usage, answers, or analytics

All logic runs entirely in the user’s web browser.

---

## Privacy and data handling

- This tool is fully static
- No data is sent to a server
- No personal information is collected or stored
- Users should not enter identifiable or confidential information

---

## How to use the tool

### Use the hosted version
If a hosted version is available, users can access the tool directly in a web browser via the published GitHub Pages link.

### Run locally
1. Download the repository (or use “Download ZIP”)
2. Open `index.html` in any modern web browser
3. No installation or configuration is required
